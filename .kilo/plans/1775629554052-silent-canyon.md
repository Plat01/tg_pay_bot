# План: Реализация системы оплаты подписки через Platega

## Цель
Реализовать полный цикл оплаты подписки:
1. Пользователь выбирает тариф
2. Выбирает метод оплаты (из доступных Platega)
3. Создается платеж на выбранную сумму
4. Пользователь оплачивает на внешнем ресурсе
5. Нажимает кнопку "Я оплатил" → проверка статуса → выдача подписки

## Флоу оплаты

### Шаг 1: Пользователь нажимает "💳 Оплатить"
```
Показать меню тарифов:
• 1 месяц — 50 ₽ (тест)
• 3 месяца — 799 ₽
• 12 месяцев — 2499 ₽
```

### Шаг 2: Пользователь выбирает тариф (например, "1 месяц")
```
callback_data: "tariff_1_month"
→ handler_tariff_selection()
→ Показать методы оплаты Platega:
  • СБП QR-код
  • Банковская карта РФ
  • Международная карта
  • Криптовалюта
  • ЕРИП
```

### Шаг 3: Пользователь выбирает метод оплаты (например, "СБП QR")
```
callback_data: "method:2" (PlategaPaymentMethod.SBP_QR = 2)
→ create_subscription_payment()
→ Create payment via Platega API
→ Save payment to database
→ Показать сообщение с кнопками:
  "✅ Платеж создан! Сумма: 50 ₽"
  [💳 Перейти к оплате] [✅ Я оплатил] [◀️ Отмена]
```

### Шаг 4: Пользователь оплачивает на внешнем ресурсе
```
Пользователь переходит по payment_url
→ Оплата на сайте Platega
```

### Шаг 5: Пользователь возвращается и нажимает "✅ Я оплатил"
```
callback_data: "confirm_payment:{payment_id}"
→ handle_confirm_payment()
→ Check payment status via Platega API
→ Если status == COMPLETED:
  - Create subscription via SubscriptionService
  - Show VPN link: "✅ Подписка активирована! Ваша VPN ссылка: ..."
→ Если status == PENDING:
  - Show: "⏳ Платеж еще не завершен. Попробуйте позже."
→ Если status == FAILED/CANCELLED:
  - Show: "❌ Платеж не прошел. Попробуйте снова."
```

## Структура данных

### Payment model (уже существует)
```python
Payment:
  - id: UUID
  - user_id: UUID
  - amount: Decimal
  - status: PaymentStatus (PENDING/COMPLETED/FAILED/CANCELLED)
  - external_id: str (Platega transaction ID)
  - payment_metadata: dict (можно добавить tariff_type)
```

### Subscription model (уже существует)
```python
Subscription:
  - id: UUID
  - user_id: UUID
  - product_id: UUID (monthly/quarterly/yearly)
  - is_active: bool
  - start_date: datetime
  - end_date: datetime
```

## Маппинг суммы → тариф

```
50 ₽   → "monthly"   (Product с subscription_type="monthly")
799 ₽  → "quarterly" (Product с subscription_type="quarterly")
2499 ₽ → "yearly"    (Product с subscription_type="yearly")
```
