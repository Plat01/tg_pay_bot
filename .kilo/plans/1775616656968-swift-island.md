# План: Реализация тестового периода с выдачей VPN ссылки

## Задача
При нажатии на кнопку "Тестовый период" показать предложение "Хочешь начать 3-дневный тестовый период?" с кнопками "Начать" и "Назад". При нажатии "Начать":
- Установить поле `is_new = False` у пользователя (триал больше недоступен)
- Создать триальную подписку на 3 дня
- Показать VPN ссылку (happ_link из product с типом "trial")

## Условия для триала (уточнено пользователем)
1. `is_new = True` (пользователь еще не использовал триал)
2. Нет активной подписки в данный момент
3. Кнопка "Тестовый период" **скрывается** из меню, если условия не выполнены

#### Добавить новый обработчик `handle_trial_activate_callback`
Активирует триал и показывает VPN ссылку:
```python
async def handle_trial_activate_callback(callback: CallbackQuery) -> None:
    """Handle ✅ Начать button - activate trial subscription."""
    async with async_session_maker() as session:
        user_service = UserService(session)
        subscription_service = SubscriptionService(session)
        product_repository = ProductRepository(session)
        
        user = await user_service.get_user_by_telegram_id(str(callback.from_user.id))
        
        if not user:
            await callback.message.edit_text(
                Texts.ERROR_NOT_REGISTERED,
                reply_markup=Keyboards.back_to_menu(),
            )
            await callback.answer()
            return
        
        # Проверить, не использовал ли пользователь триал
        if not user.is_new:
            await callback.message.edit_text(
                Texts.TRIAL_ALREADY_USED,
                parse_mode="HTML",
                reply_markup=Keyboards.buy_subscription(),
            )
            await callback.answer()
            return
        
        # Получить триальный продукт
        trial_product = await product_repository.get_product_by_subscription_type("trial")
        if not trial_product:
            await callback.message.edit_text(
                "❌ Триальный продукт не найден. Обратитесь в поддержку.",
                parse_mode="HTML",
                reply_markup=Keyboards.back_to_menu(),
            )
            await callback.answer()
            return
        
        # Создать триальную подписку
        subscription = await subscription_service.activate_trial(user.id)
        
        # Отметить, что триал использован
        await user_service.mark_trial_used(user)
        
        # Показать сообщение с VPN ссылкой
        await callback.message.edit_text(
            Texts.TRIAL_ACTIVATED.format(vpn_link=trial_product.happ_link),
            parse_mode="HTML",
            reply_markup=Keyboards.back_to_menu(),
        )
        await callback.answer()
        
        logger.info(f"User {user.telegram_id} activated trial subscription")
```

## Тестирование

   - Кнопка "Тестовый период" скрыта в меню
   
4. VPN ссылка:
   - Правильно отображается и копируется при нажатии
   - Берется из product с subscription_type="trial"
